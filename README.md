# A2ui example

Simple app that presents all features of A2ui. 

# Examples live 

If you are interested only about final effect see our demo live at: [http://a2ui-examples.codeweaver.pl](http://a2ui-examples.codeweaver.pl)

# Installation

Run:

```shell
npm install
```

Then:

```shell
ng serve
```

And we are good to go !
